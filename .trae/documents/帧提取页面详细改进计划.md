# 帧提取页面详细改进计划

## 问题分析

根据您的反馈，我已经识别了以下具体问题：

1. **帧拖动问题**：
   - 手动拖动帧时，帧会重合或重叠
   - 帧会被不小心拖动超出放置区域，导致被裁剪
   - 没有边界限制和碰撞检测

2. **间距问题**：
   - `gridPadding` 设置的间距在实际操作中没有生效
   - 间距只在自动排列时有效，在手动拖动和导出时没有应用
   - 缺少间距的视觉反馈

3. **导出流程问题**：
   - 导出时直接就导出了，没有预览功能
   - 用户无法在导出前检查最终效果
   - 缺少确认步骤，可能导致不满意的结果

## 解决方案和实施步骤

### 第一步：改进帧拖动功能

1. **添加边界限制**：
   - 在 `handleExtractTargetMove` 函数中添加代码，确保帧不能超出 `targetFrames` 容器边界
   - 计算帧位置时，确保帧的左上角和右下角都在容器内
   - 如果帧超出边界，将其位置限制在容器边缘

2. **添加碰撞检测**：
   - 在拖动过程中检查当前帧是否与其他帧重叠
   - 如果检测到碰撞，提供视觉反馈（如红色边框）
   - 在松开鼠标时，如果仍然有重叠，将帧自动调整到不重叠的位置

3. **添加网格吸附**（可选增强）：
   - 让帧在拖动时自动对齐到网格
   - 这样可以更好地控制布局

### 第二步：确保间距正确应用

1. **修改 `autoArrangeExtractTarget` 函数**：
   - 确保间距在计算帧位置时被正确应用
   - 在水平和垂直方向上都应用间距
   - 添加视觉反馈，显示间距区域

2. **修改 `exportAsSpriteSheet` 函数**：
   - 确保在导出精灵表时也应用间距
   - 在计算画布大小时，为每个帧添加间距
   - 在绘制帧时，确保帧之间有足够的间距

3. **修改 `sendToFrameEdit` 函数**：
   - 确保在发送到帧编辑页面时也保持间距
   - 在计算组合画布大小时应用间距

### 第三步：添加导出预览和确认功能

1. **创建新的导出预览模态框**：
   - 在 `index.html` 中添加新的模态框结构
   - 模态框包含：
     - 标题和说明
     - 预览画布，显示最终的精灵表效果
     - 确认和取消按钮
     - （可选）导出格式选择

2. **修改导出流程**：
   - 原来的 `executeExportOption` 函数在用户选择导出后直接执行
   - 修改为先显示预览，等待用户确认后再执行实际的导出
   - 如果用户选择取消，返回到原来的界面

3. **实现预览生成功能**：
   - 创建一个函数，在导出前生成预览画布
   - 确保预览和实际导出的结果一致
   - 在预览中也应用间距设置

## 修改文件清单

1. **index.html**：
   - 添加新的导出预览模态框
   - 修改模态框结构

2. **js/frame-extract.js**：
   - 修改 `handleExtractTargetMove`：添加边界限制和碰撞检测

### 1. 改进 `handleExtractTargetMove` 函数

```javascript
function handleExtractTargetMove(e) {
  if (!extractTargetInteraction || extractTargetInteraction.type !== 'move') return;
  
  const target = window.extractTargets.find(t => t.id === extractTargetInteraction.targetId);
  if (!target) return;
  
  const coords = extractTargetCoords(e);
  const targetArea = document.getElementById('targetFrames');
  
  // 计算目标位置
  let newX = extractTargetInteraction.frameX + coords.x - extractTargetInteraction.startX;
  let newY = extractTargetInteraction.frameY + coords.y - extractTargetInteraction.startY;
  
  // 添加边界限制
  newX = Math.max(0, newX);
  newY = Math.max(0, newY);
  
  // 确保帧不超出容器边界
  newX = Math.min(newX, 800 - target.w); // 假设容器最大宽度为800px，根据实际情况调整
  // 这里需要根据实际容器高度调整Y轴的边界
  
  // 添加碰撞检测
  const otherTargets = window.extractTargets.filter(t => t.id !== target.id);
  let hasCollision = false;
  
  // 检查碰撞
  for (const other of otherTargets) {
    if (newX < other.x + other.w &&
        newX + target.w > other.x &&
        newY < other.y + other.h &&
        newY + target.h > other.y) {
      hasCollision = true;
      break;
    }
  }
  
  // 更新位置
  target.x = newX;
  target.y = newY;
  
  // 渲染并更新视觉反馈
  renderExtractTargets();
  
  // 如果有碰撞，添加视觉提示
  const targetElement = document.querySelector(`[onclick="removeExtractTarget('${target.id}')"]`).parentElement;
  if (targetElement) {
    if (hasCollision) {
      targetElement.style.borderColor = 'var(--danger)';
      targetElement.style.borderWidth = '3px';
    } else {
      targetElement.style.borderColor = 'var(--accent)';
      targetElement.style.borderWidth = '2px';
    }
  }
}
```

### 2. 改进 `autoArrangeExtractTarget` 函数

```javascript
function autoArrangeExtractTarget() {
  const gridSize = parseInt(document.getElementById('gridSize').value) || 32;
  const padding = parseInt(document.getElementById('gridPadding').value) || 2;
  
  let x = padding;
  let y = padding;
  let maxRowHeight = 0;
  
  window.extractTargets.forEach((target, index) => {
    const gridWidth = Math.ceil(target.w / gridSize);
    const gridHeight = Math.ceil(target.h / gridSize);
    
    // 检查是否需要换行
    if (x + (gridWidth * gridSize) + padding > 800) {
      x = padding;
      y += maxRowHeight + padding;
      maxRowHeight = 0;
    }
    
    target.x = x;
    target.y = y;
    
    // 更新当前行的最大高度
    maxRowHeight = Math.max(maxRowHeight, gridHeight * gridSize);
    
    // 为下一帧计算位置，包括间距
    x += (gridWidth * gridSize) + padding;
  });
  
  renderExtractTargets();
}
```

### 3. 添加导出预览功能

```javascript
// 新增函数：显示导出预览
function showExportPreview() {
  const padding = parseInt(document.getElementById('gridPadding').value) || 2;
  
  // 计算画布大小
  let maxX = 0;
  let maxY = 0;
  window.extractTargets.forEach(target => {
    maxX = Math.max(maxX, target.x + target.w + padding);
    maxY = Math.max(maxY, target.y + target.h + padding);
  });
  
  // 创建预览画布
  const previewCanvas = document.createElement('canvas');
  previewCanvas.width = maxX;
  previewCanvas.height = maxY;
  const ctx = previewCanvas.getContext('2d');
  
  // 填充透明背景
  ctx.fillStyle = 'rgba(0,0,0,0)';
  ctx.fillRect(0, 0, previewCanvas.width, previewCanvas.height);
  
  // 绘制所有帧到预览画布
  let loadedCount = 0;
  window.extractTargets.forEach(target => {
    const img = new Image();
    img.onload = () => {
      ctx.drawImage(img, target.x, target.y);
      loadedCount++;
      
      // 当所有帧都绘制完成时，显示预览
      if (loadedCount === window.extractTargets.length) {
        // 更新预览模态框
        const previewContainer = document.getElementById('exportPreviewCanvas');
        if (previewContainer) {
          previewContainer.innerHTML = '';
          previewContainer.appendChild(previewCanvas);
        }
        
        // 显示预览模态框
        document.getElementById('exportPreviewModal').classList.add('show');
      }
    };
    img.src = target.imgData;
  });
}

// 修改执行导出选项函数
function executeExportOption() {
  console.log('executeExportOption called');
  console.log('selectedExportOption:', selectedExportOption);
  closeModal('extractExportModal');
  
  if (selectedExportOption === 'spriteSheet') {
    // 先显示预览，不直接导出
    showExportPreview();
  } else if (selectedExportOption === 'frameEdit') {
    // 发送到帧编辑页面
    sendToFrameEdit();
  }
}

// 新增函数：确认导出
function confirmExport() {
  // 关闭预览模态框
  document.getElementById('exportPreviewModal').classList.remove('show');
  
  // 执行实际的导出
  exportAsSpriteSheet();
}

// 修改 exportAsSpriteSheet 函数，确保间距正确应用
function exportAsSpriteSheet() {
  const padding = parseInt(document.getElementById('gridPadding').value) || 2;
  
  // 计算画布大小，确保每个帧周围都有间距
  let maxX = 0;
  let maxY = 0;
  window.extractTargets.forEach(target => {
    maxX = Math.max(maxX, target.x + target.w + padding);
    maxY = Math.max(maxY, target.y + target.h + padding);
  });
  
  // 创建画布
  const canvas = document.createElement('canvas');
  canvas.width = maxX;
  canvas.height = maxY;
  const ctx = canvas.getContext('2d');
  
  // 填充透明背景
  ctx.fillStyle = 'rgba(0,0,0,0)';
  ctx.fillRect(0, 0, canvas.width, canvas.height);
  
  // 重置所有target的loaded状态
  window.extractTargets.forEach(target => {
    target.loaded = false;
  });
  
  // 绘制所有帧
  window.extractTargets.forEach(target => {
    const img = new Image();
    img.onload = () => {
      ctx.drawImage(img, target.x, target.y);
      target.loaded = true;
      
      // 检查是否所有帧都已绘制
      const loadedCount = window.extractTargets.filter(t => t.loaded).length;
      if (loadedCount === window.extractTargets.length) {
        // 所有帧都绘制完成，导出
        exportCanvasAsImage(canvas, 'sprite_sheet');
      }
    };
    img.onerror = () => {
      console.error('Error loading target for export');
      target.loaded = true;
      
      // 检查是否所有目标都已处理
      const loadedCount = window.extractTargets.filter(t => t.loaded).length;
      if (loadedCount === window.extractTargets.length) {
        exportCanvasAsImage(canvas, 'sprite_sheet');
      }
    };
    img.src = target.imgData;
  });
}
```

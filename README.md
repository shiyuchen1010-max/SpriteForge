# SpriteForge - 精灵表编辑器

一款零依赖、浏览器端运行的精灵表（Sprite Sheet）编辑工具，专为像素艺术和 2D 游戏开发设计。

## ✨ 特性

- **手动框选抠帧** — 在精灵表上自由框选每一帧，支持自动检测和手动模式
- **智能裁剪** — 自动去除每帧的透明边缘，减少冗余像素
- **锚点微调** — 逐帧设定锚点位置，支持参考线、颜色切换、快捷键微调
- **防抖动对齐** — 统一帧尺寸 + 重心对齐 + 冗余边距，确保动画播放不晃动
- **实时动画预览** — 逐帧查看、播放/暂停、帧率调节
- **帧顺序编辑** — 上移/下移调整帧的播放顺序
- **多格式导出** — 支持 JSON、TexturePacker XML、CSS Spritesheet、Packer TXT
- **ZIP 打包导出** — 一键导出单帧图片 + 精灵表 + 元数据
- **零安装** — 单个 HTML 文件，浏览器打开即用

## 🚀 快速开始

1. 下载 `index.html`
2. 用浏览器打开
3. 拖入或选择精灵表图片
4. 开始编辑！

## 📖 使用流程

```
导入精灵表 → 放大到合适大小 → 手动框选每一帧 → 帧对齐与导出
                                                    ↓
                                            设置对齐参数
                                            设定锚点（可选）
                                            预览动画效果
                                            命名并导出
```

### 详细步骤

1. **导入** — 拖拽图片到页面或点击"打开图片"
2. **抠帧** — 在精灵表上拖拽框选每一帧，支持网格吸附
3. **帧对齐与导出** — 点击顶部"帧对齐与导出"按钮：
   - **Step 1**：选择对齐模式（水平/垂直）、透明度阈值，设定锚点
   - **Step 2**：预览对齐后的动画，微调锚点位置
   - **Step 3**：命名、选择导出格式，下载

## ⌨️ 快捷键

| 快捷键 | 功能 |
|--------|------|
| `Space` | 播放/暂停动画 |
| `Delete` / `Backspace` | 删除选中帧 |
| `Ctrl+A` | 全选帧 |
| `Escape` | 关闭弹窗 |
| `←` `→` `↑` `↓` | 微调锚点位置（长按连续） |
| `Ctrl+←` `Ctrl+→` | 切换上一帧/下一帧 |

## 📦 导出格式

| 格式 | 扩展名 | 适用场景 |
|------|--------|---------|
| JSON (通用) | `.json` | 所有引擎通用 |
| TexturePacker XML | `.xml` | Unity、Cocos2d、Phaser |
| CSS Spritesheet | `.css` | Web 前端开发 |
| Packer TXT | `.txt` | 简单文本格式 |

### 在 Godot 中使用

1. 导出选择 **JSON (通用)** 格式
2. 将 `.png` 精灵表导入 Godot
3. 创建 `AnimatedSprite2D` 节点
4. 设置 `hframes` = JSON 中的 `cols`，`vframes` = `rows`
5. Offset 设为 **Center** 或 **Bottom Center**
6. 完成！

## 🛠️ 技术栈

- 纯 HTML + CSS + JavaScript
- 无任何外部依赖（JSZip 通过 CDN 按需加载）
- Canvas API 进行图像处理
- 支持所有现代浏览器
